Objetos y Clases

Objetos

  • Base de la Programación Orientada a Objetos es el objeto
  • Los objetos tienen una serie de características y un comportamiento
  • Campos o Atributos: variables - boolean, int double, char, Persona, Auto, Estudiante
  • Rutinas o Métodos: lleva a cabo una determinada acción

Objetos y Clases

Clases

  • Conjunto de objetos que comparten una estructura y un comportamiento comunes
  • Considerado como un tipo NO PRIMITIVO
  • Plantilla de objetos
  • Identificadores simples, descriptivos y sustantivos

Objetos y Clases

Instancias

  • Una instancia es un elemento tangible
  • Todos los objetos empleados en un programa han de pertenecer a una clase determinada
  • Instancia de una clase predifina en JAVA-PYTHON o declarada por el usuario
  • El acceso a los atributos se realiza a través del operador punto

  • Comienzan con la palabra public - disponible al público
  • Tipo de valor de retorno (void - String - int)
  • Letra minúscula
  • Puede tener una lista de parámetros

Variables de Instancia

  • Clase: uno o más métodos que manipulan los atributos de un objeto específico
  • Atributos: campos - se declaran dentro de la declaración de una clase ... pero fuera de la declaración de los métodos
  • Un objeto cuando está instanciado hace una copia de los atributos: variable de instancia

Establecer (set) y Obtener (get)

  • Se usa variables globales en una clase ...
  • puedan ser usadas o modificadas en cualquier instante
  • fuera de la declaración de los métodos
  • Un objeto cuando está instanciado hace una copia de los atributos: variable de instancia
 

 public class Calificaciones2 {
    private String nombreCurso;

    public void establecerNombreCurso(String nombre){
        nombreCurso = nombre;
    }

    public String obtenerNombreCurso(){
        return nombreCurso;
    }

    public void mostrarMensaje(){
        System.out.printf("El nombre del curso es %s", obtenerNombreCurso());
    }
}

public - private

"La palabra clave private es un modifi cador de acceso. Las variables o los méto- dos declarados con el modifi cador de acceso private son accesibles sólo para los métodos de la clase en la que se declaran" - Ocultamiento de datos

Evitar que se modifiquen los atributos de las instancias de las clases (objetos) por accidente

tipo primitivos - tipos por referencia

  • Tipos primitivos: byte, char, int, double
  • Tipos por referencia: todos lo que NO SON PRIMITIVOS

constructores

  • Inicializar variables al momento de crear al objeto
  • Usar método especial llamado constructor
  • El nombre del constructor debe ser el mismo de la clase
  • No retornan valores
